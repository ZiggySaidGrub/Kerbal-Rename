This is a short guide on how to use the Kerbal Rename tool.
1.Enter the path of your KSP save folder in the file labeled "path.txt". This only needs to be done once. (eg. C:\Program Files (x86)\Steam\steamapps\common\Kerbal Space Program\saves\)
2.Run the program.
3.Select the save you want to edit.
4.Enter the *FULL* name of the Kerbal you want to edit. (eg. >>>Jebediah Kerman)
5.Enter the new name of the Kerbal.
6.Relaunch KSP with your renamed Kerbal!
